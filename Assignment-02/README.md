# Assignment 02

## Aim
To design and develop a dynamic E-Commerce/Fashion Store web application using HTML, CSS, JavaScript, and Bootstrap that provides an interactive shopping experience.

## Technologies Used
- HTML
- CSS
- JavaScript
- Bootstrap 5 (CDN)

## Steps Performed
1. Designed a responsive user interface using HTML and Bootstrap.
2. Styled the application using CSS with responsive layouts and hover effects.
3. Created product cards displaying images, names, prices, and action buttons.
4. Implemented category-based product filtering using JavaScript.
5. Added dynamic shopping cart functionality with subtotal calculation.
6. Implemented wishlist functionality for saving favorite products.
7. Displayed toast notifications for user actions such as adding products to the cart or wishlist.
8. Tested the application on different screen sizes and captured output screenshots.

## Features
- Responsive Navigation Bar
- Product Category Filtering
- Dynamic Product Cards
- Shopping Cart with Subtotal
- Wishlist Management
- Toast Notifications
- Responsive Design using Bootstrap
- Interactive User Experience

## Output
The dynamic Fashion Store/E-Commerce web application successfully displays products, allows users to filter categories, add products to the cart and wishlist, updates the cart dynamically, and provides a responsive and interactive shopping experience. Output screenshots are available in the `output-screenshots` folder.
