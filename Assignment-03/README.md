# Assignment 03

## Aim
To design and develop a full-stack Expense Tracker web application using React.js, Express.js, Node.js, and MongoDB with JWT authentication for secure user access and transaction management.

## Technologies Used
- React.js
- Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB
- JSON Web Token (JWT)

## Steps Performed
1. Set up the frontend using React.js and Vite.
2. Designed a responsive user interface using Tailwind CSS.
3. Developed RESTful APIs using Express.js and Node.js.
4. Connected the backend with MongoDB for storing user and transaction data.
5. Implemented JWT-based authentication for secure login and registration.
6. Added functionalities to create, view, update, and delete expense transactions.
7. Displayed transaction history and dashboard analytics dynamically.
8. Tested the application and captured output screenshots.

## Features
- User Registration and Login
- JWT Authentication
- Add, Update, and Delete Transactions
- Expense and Income Tracking
- Dashboard with Transaction Summary
- MongoDB Database Integration
- Responsive User Interface
- Secure RESTful APIs

## Output
The Expense Tracker web application successfully allows users to securely manage income and expense records using React.js, Express.js, Node.js, and MongoDB. The application provides authentication, dynamic transaction management, and an interactive dashboard. Output screenshots are available in the `output-screenshots` folder.
