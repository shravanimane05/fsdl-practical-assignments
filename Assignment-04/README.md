# Assignment 04

## Aim
To design and develop a web-based Attendance Management and Warning Letter Generation System using React.js, Node.js, Express.js, and modern JavaScript libraries to automate attendance analysis and student notification.

## Technologies Used
- React.js
- Node.js
- Express.js
- Vite
- Tailwind CSS
- XLSX
- Recharts
- jsPDF
- JSZip
- File Saver
- LocalStorage

## Steps Performed
1. Developed the frontend using React.js with Vite and Tailwind CSS.
2. Implemented a login system for user authentication.
3. Added functionality to upload and process Excel attendance sheets using the XLSX library.
4. Extracted student attendance data and identified students with attendance below 75%.
5. Displayed attendance statistics and charts using Recharts.
6. Generated warning letters in PDF format using jsPDF.
7. Implemented bulk PDF download using JSZip and File Saver.
8. Added email notification functionality for sending warning messages to parents or guardians.
9. Stored upload history and session information using LocalStorage.
10. Tested the application and captured output screenshots.

## Features
- Secure Login System
- Excel File Upload and Processing
- Automatic Attendance Analysis
- Low Attendance Student Identification
- Interactive Dashboard
- Attendance Visualization with Charts
- PDF Warning Letter Generation
- Bulk ZIP Download of Warning Letters
- Email Notification Support
- Upload History Management
- Responsive User Interface

## Output
The Attendance Management and Warning Letter Generation System successfully automates attendance analysis by processing Excel files, identifying students with attendance below 75%, generating PDF warning letters, providing email notification functionality, and displaying attendance statistics through an interactive dashboard. Output screenshots are available in the `output-screenshots` folder.
